## Схема базы данных для Dating бота

Этот файл демонстрирует основные сущности и таблицы БД проекта.  
README содержит общее ТЗ, а здесь — детализированная структура таблиц.

---

### Таблица `users`

| Поле          | Тип            | Описание                                 |
|--------------|----------------|------------------------------------------|
| `id`         | PK, UUID/INT   | Уникальный идентификатор пользователя   |
| `telegram_id`| BIGINT, UNIQUE | ID пользователя в Telegram              |
| `username`   | VARCHAR        | Telegram username (опционально)         |
| `first_name` | VARCHAR        | Имя                                      |
| `language`   | VARCHAR(10)    | Язык интерфейса                          |
| `created_at` | TIMESTAMP      | Дата регистрации                         |
| `updated_at` | TIMESTAMP      | Дата последнего обновления              |

---

### Таблица `profiles`

| Поле           | Тип              | Описание                                  |
|---------------|------------------|-------------------------------------------|
| `id`          | PK, UUID/INT     | Уникальный идентификатор анкеты          |
| `user_id`     | FK → `users.id`  | Владелец анкеты                          |
| `age`         | INT              | Возраст                                   |
| `gender`      | VARCHAR          | Пол                                       |
| `city`        | VARCHAR          | Город/регион                              |
| `interests`   | TEXT/JSON        | Список интересов                          |
| `about`       | TEXT             | Описание «о себе»                         |
| `pref_gender` | VARCHAR          | Искомый пол                               |
| `pref_age_min`| INT              | Минимальный возраст партнёра             |
| `pref_age_max`| INT              | Максимальный возраст партнёра            |
| `pref_city`   | VARCHAR          | Город/регион для поиска                  |
| `is_active`   | BOOL             | Активна ли анкета                         |
| `created_at`  | TIMESTAMP        | Дата создания                             |
| `updated_at`  | TIMESTAMP        | Дата последнего обновления               |

---

### Таблица `profile_photos`

| Поле        | Тип               | Описание                                  |
|------------|-------------------|-------------------------------------------|
| `id`       | PK, UUID/INT      | Уникальный идентификатор записи          |
| `profile_id`| FK → `profiles.id`| Ссылка на анкету                         |
| `url`      | VARCHAR/TEXT      | S3/Minio URL изображения                  |
| `is_main`  | BOOL              | Является ли фото основным                 |
| `created_at`| TIMESTAMP        | Дата загрузки                             |

---

### Таблица `likes`

| Поле           | Тип                | Описание                                 |
|---------------|--------------------|------------------------------------------|
| `id`          | PK, UUID/INT       | Уникальный идентификатор                |
| `from_user_id`| FK → `users.id`    | Кто поставил лайк/пропуск               |
| `to_profile_id`| FK → `profiles.id`| Чья анкета                              |
| `action`      | ENUM(like, skip)   | Тип действия                             |
| `created_at`  | TIMESTAMP          | Время действия                           |

---

### Таблица `matches`

| Поле        | Тип               | Описание                                  |
|------------|-------------------|-------------------------------------------|
| `id`       | PK, UUID/INT      | Уникальный идентификатор                 |
| `user1_id` | FK → `users.id`   | Первый участник мэтча                    |
| `user2_id` | FK → `users.id`   | Второй участник мэтча                    |
| `created_at`| TIMESTAMP        | Время образования мэтча                  |

---

### Таблица `dialogs`

| Поле        | Тип               | Описание                                  |
|------------|-------------------|-------------------------------------------|
| `id`       | PK, UUID/INT      | Уникальный идентификатор диалога         |
| `match_id` | FK → `matches.id` | Мэтч, к которому относится диалог        |
| `created_at`| TIMESTAMP        | Дата создания диалога                    |

---

### Таблица `messages`

| Поле        | Тип               | Описание                                  |
|------------|-------------------|-------------------------------------------|
| `id`       | PK, UUID/INT      | Уникальный идентификатор сообщения       |
| `dialog_id`| FK → `dialogs.id` | Диалог                                   |
| `sender_id`| FK → `users.id`   | Отправитель                              |
| `text`     | TEXT              | Текст сообщения                          |
| `created_at`| TIMESTAMP        | Время отправки                           |

---

### Таблица `ratings`

| Поле               | Тип               | Описание                                 |
|--------------------|-------------------|------------------------------------------|
| `id`               | PK, UUID/INT      | Уникальный идентификатор                |
| `profile_id`       | FK → `profiles.id`| Анкета                                  |
| `primary_score`    | NUMERIC/DOUBLE    | Первичный рейтинг                       |
| `behavioral_score` | NUMERIC/DOUBLE    | Поведенческий рейтинг                   |
| `combined_score`   | NUMERIC/DOUBLE    | Комбинированный рейтинг                 |
| `updated_at`       | TIMESTAMP         | Время последнего пересчёта              |

---

### Таблица `referrals` (опционально)

| Поле        | Тип               | Описание                                  |
|------------|-------------------|-------------------------------------------|
| `id`       | PK, UUID/INT      | Уникальный идентификатор                 |
| `inviter_id`| FK → `users.id`  | Пригласивший пользователь                |
| `invitee_id`| FK → `users.id`  | Приглашённый пользователь                |
| `created_at`| TIMESTAMP        | Дата регистрации по рефералке            |
